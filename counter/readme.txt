设计一个8位的计数器
1 两个8位的数据输入，分别为A[A7-A0]和B[B7-B0].
2 两个功能选择输入，分别为SEL1和SEL2。
功能对应如下：
SEL2   SEL1         功能
0       0           8位无符号加法器
0       1           8位无符号比较器
1       0           8位计数器
1       1           无用
3 其它输入
CLK             时钟
RST             复位： 对计数器和加法器功能有效。高电平有效
Carry           加法器进位。在计数器功能是也用作RST信号
4 三
